# 시스템 클럭 & 틱
- 시스템 클럭: 50MHz
1. 시스템 클럭은 모든 모듈(클럭 분주기, 타이머, LCD 컨트롤러, 피에조 컨트롤러, FSM 등) 모든 clk 포트에 동일하게 전달
2. 분주기에서는 카운터로 50,000번을 세어서 1ms마다 딱 1클럭 동안만 1이 되는 tick 신호를 만듦
3. 게임 모듈들에는 50MHz 클럭을 집어 넣으면서 분주기에서 온 tick 신호를 사용함.
always @(posedge clk) 안에서 tick 신호가 1일 때만 처리를 하면 됨
4. LCD나 피에조같은 고속 모듈들에서는 tick 신호 말고 그대로 고속 신호인 50MHz 사용하면 됨

## 시뮬레이션 방법
1. 테스트벤치 파일 작성
2. 왼쪽 메뉴에서 Add Sources로 시뮬레이션 추가
3. SIMULATION 탭에서 Run Simulation
